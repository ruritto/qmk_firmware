# 夏休み宿題管理アプリ（中3・2026年夏）

`夏休みタスクデータ.json` をシードデータとして組み込んだ、中学3年生の夏休み用の宿題管理Webアプリです。
フレームワークなしのプレーンな HTML / CSS / JavaScript 1ファイルで動作します。

## 起動方法

`index.html` をブラウザで開くだけで動きます（シードデータはHTML内に埋め込み済みのため、`file://` でもそのまま動作します）。

ローカルサーバーを使う場合:

```sh
cd summer-homework-app
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

## 機能

- **日ごとビュー**: 1日分のタスク・塾の授業・塾復習・締切をまとめて表示。前日/翌日ナビと「今日」ボタン付き。`excluded: true` の日は「📅 予定あり」バッジと斜線背景で表示。
- **週ごとビュー**: 日曜〜土曜の1週間単位で表示。前週/次週ナビと週の達成率をヘッダーに表示。日付をタップすると日ごとビューへ移動。
- **チェックボックス**: school / exam / deadline のタスクと塾復習（juku_review）にチェックボックスを表示。info（試験当日などの参考情報）はチェック対象外。
- **色分け**: 学校の宿題=オレンジ系 `#B8420C`、期末試験=ネイビー系 `#2B4570`、締切=赤系 `#C1440E`、塾=グレー系 `#6E6A56`、背景 `#F5F1E6`。
- **未完了一覧**: 全期間の未完了タスクを横断表示。締切カテゴリを最優先、以降は日付順にソート。期日3日以内は赤枠+「あとn日」チップで強調。カテゴリフィルタ付き。
- **完了ログ**: いつ・どのタスクを完了にしたかを日時付きで記録。**チェックを外した場合は完了記録を消さず「取り消し」として履歴に追記**します。
- **達成率**: 全体（ドーナツチャート）・週ごと・教科ごとの達成率を表示。塾復習を達成率に含めるかはトグルで切替可能（デフォルトON）。
- **要確認メモ**: シードの `open_questions` を「達成率」タブ内に表示。確認済みチェック・編集・削除・追加が可能。
- **この日の予定**: 日ごとビューの日付の下に、塾の授業（時間割画像から読み取った授業時間つき）と部活「夏休み中の練習」（基本9:00〜12:00、7/29コンクール・8/1大向フェスティバルなどの特別日を含む）を表示。
- **宿題の追加**: 日ごとビューの「＋宿題・タスクを追加」から、後から発生した宿題を任意の日付・カテゴリで追加可能（追加したタスクは✕で削除可能。localStorageに保存）。
- **褒めメッセージ**: その日のチェック対象タスクを全部完了すると「🎉全部完了！やったね！」などのメッセージを表示。
- **今週の残りタスク**: 日ごとビューの下部に、同じ週のほかの日の未完了タスクを色違いのカードでまとめて表示（そこから直接チェックも可能）。
- **タスクの編集・移動**: 各タスクの「✎」ボタン（週ごと・日ごと・未完了ビュー共通）から、タイトル変更と別の日への移動ができる（移動したタスクには「📅 元の日から移動」の表示が付く。シード由来のタスクはJSONを書き換えず、変更内容をlocalStorageに上書き保存）。
- **予定の追加**: 日ごとビューの「この日の予定」欄の「＋予定を追加」から、内容・時間・日付を指定して予定を登録できる（✎で内容・時間の変更、日付移動、削除も可能。週ごとビューにもチップ表示）。

## データの扱い

- 初回起動時のみシードデータ（HTMLに埋め込んだ `夏休みタスクデータ.json` の内容）から初期化します。
- 完了状態・完了ログ・要確認メモ・設定はブラウザの **localStorage**（キー: `natsu-shukudai-v1`）に保存され、再読み込みしても保持されます。JSONファイル自体は書き換えません。
- 保存は端末（ブラウザ）ごとです。別の端末では完了状態は共有されません。

## Vercelでの配信（単独プロジェクト）

vr-task-app には独自の認証（許可ドメインのGoogleアカウントのみ）が組み込まれているため、このアプリは **vr-task-app とは別のVercelプロジェクト**として配信します。

初回のみ、Vercelダッシュボードで次の設定をしてください：

1. Vercel ダッシュボード → 「Add New…」→「Project」
2. このリポジトリ（qmk_firmware）を「Import」
3. **Root Directory** の「Edit」を押して `summer-homework-app` を選択
4. Framework Preset は「Other」のまま、Build Command / Output Directory は空のまま「Deploy」

以降は master への push のたびに自動デプロイされ、`https://<プロジェクト名>.vercel.app/` でログイン不要でアクセスできます（`index.html` がそのままトップページになります）。

## 家族からのアクセスについて

このアプリは Claude の Artifact としても公開できます（claude.ai は Googleアカウントでのログインに対応）。
公開したArtifactのリンクを家族に共有すれば、タブレット等のブラウザからアクセスできます。
ただしチェック状態は端末ごとの保存となるため、家族間でのリアルタイム同期はありません。
